# GHAZI Stocks & Options Radar

مشروع مجاني يجمع **المحفزات الرسمية + ترتيب الأسهم + اختيار أفضل عقود الخيارات**.
يبدأ من SEC وFDA والأخبار، ثم يحلل السهم فنيًا وقطاعيًا، وبعدها يفحص العقود فقط
للأسهم الأعلى تقييمًا.

> البيانات المجانية لا تثبت Smart Money Sweep لحظيًا ولا تضمن الربح. النظام يعرض
> جودة المصدر وحداثته، ويستخدم `aggressor_proxy` بوصفه مؤشر نشاط تقريبيًا فقط.

## طريقة التشغيل المعتمدة

المشروع **Dashboard Only**:

- يعرض النتائج في صفحة الويب.
- يحفظ سجل الإشارات والنتائج والمعايرة داخل GitHub.
- لا يرسل بريدًا أو Telegram أو Discord.
- لا ينفذ أوامر تداول آلية.

## المزايا

- صفحة للأسهم المرشحة مع الدخول والأهداف والإبطال.
- صفحة لأفضل العقود المرتبطة بالأسهم الأعلى تقييمًا.
- صفحة لمحفزات SEC EDGAR وopenFDA والأخبار.
- قراءة 8-K و6-K و13D وForm 4 والكلمات الجوهرية داخل المستندات.
- اكتشاف الطروحات والتخفيف والأخبار السلبية وخصمها من الدرجة.
- تحليل EMA 9/21/50/200 وRSI وMACD وATR والحجم النسبي واختراق 20 يومًا.
- قوة نسبية مقابل القطاع والسوق.
- فلترة DTE والسيولة وVol/OI والـspread والدلتا وIV مقابل realized volatility.
- بوابة جودة ترفض البيانات القديمة والعقود غير القياسية والسبريد غير الصالح.
- سجل دائم للإشارات وMFE وMAE ونتائج الأهداف والوقف.
- صفحة للفرص المرفوضة وأسباب الرفض.
- بوابة معايرة لا تسمح بتغيير الأوزان قبل اكتمال العينة.
- GitHub Actions مجدول لتحديث البيانات.

## المصادر المجانية

1. SEC EDGAR: إفصاحات رسمية ولا يحتاج مفتاح API.
2. openFDA / Drugs@FDA: يعمل دون مفتاح ضمن حد عام، والمفتاح المجاني اختياري.
3. Yahoo/yfinance: أسعار وشارت وسلاسل خيارات وأخبار، مع تنبيه أنه مصدر غير رسمي.
4. Tradier أو MarketData.app أو Alpaca اختيارية عند إضافة مفتاح؛ المشروع يعود إلى Yahoo عند غيابها.

## التشغيل المحلي

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate

pip install -r requirements.txt
copy .env.example .env   # Windows
# cp .env.example .env   # macOS/Linux
python main.py --mode all --top 15
```

خدمة الصفحة الثابتة:

```bash
python main.py
```

وعند وجود متغير `PORT` يشغّل `main.py` خادم HTTP على `0.0.0.0:$PORT` لخدمة مجلد `public/`.

## الأقسام داخل الصفحة

1. **الأسهم المرشحة:** درجة السهم، جهة CALL/PUT، منطقة الدخول، هدفان، الإبطال، RSI، RVOL، القطاع والمحفز.
2. **أفضل العقود:** تاريخ الانتهاء، Strike، Call/Put، Vol/OI، IV، Delta، Spread، الدخول والأهداف والوقف.
3. **محفزات SEC وFDA:** المصدر والرابط والثقة والقيمة وغرض الإفصاح.
4. **الفرص المرفوضة:** سبب رفض السهم أو العقد.
5. **النتائج والمعايرة:** العينة المسعرة وMFE وMAE ومعدلات الأهداف والوقف.
6. **حالة التشغيل:** المصدر، جلسة السوق، حجم الكون وأخطاء المصادر.

## اختيار مصدر العقود

```dotenv
OPTIONS_PROVIDER=auto
```

الأولوية في `auto`:

1. Tradier عند وجود `TRADIER_TOKEN`.
2. MarketData.app عند وجود `MARKETDATA_TOKEN`.
3. Yahoo/yfinance كخيار بلا مفتاح.

Alpaca يثري bid/ask وGreeks عند توفره، ولا يستبدل volume/OI.

## SEC وFDA

SEC يطلب من البرامج تعريف التطبيق ووسيلة تواصل:

```dotenv
SEC_USER_AGENT=GHAZI Options Radar your-email@example.com
```

openFDA يعمل دون مفتاح ضمن الحد العام، ويمكن إضافة مفتاح مجاني لرفع الحد:

```dotenv
OPENFDA_API_KEY=
```

## GitHub Actions

الـworkflow يحدث الصفحة في الدقيقتين `07` و`37` خلال نافذة السوق، ويأخذ لقطة قبل الجلسة.
التطبيق يتخطى العطلات والأوقات خارج الجلسة في التشغيلات الدورية. قد تتأخر جدولة GitHub Actions،
ولذلك ليست بديلًا عن بيانات Tick لحظية أو خادم تداول منخفض الكمون.

عند اكتمال 100 إشارة مسعرة يفتح النظام Issue للمراجعة المستقلة، ولا يغيّر الأوزان تلقائيًا.

## منطق التقييم

### السهم

- الاتجاه والمتوسطات وMACD وRSI والحجم والاختراق أو الكسر.
- السيولة بالدولار.
- محفز رسمي إيجابي أو سلبي.
- توافق SPY وQQQ وVIX والقطاع.
- خصم التشبع والامتداد والطروحات والتخفيف.

### العقد

- Volume وOpen Interest وVol/OI والـspread.
- Delta/Gamma وIV مقارنة بالتذبذب المحقق.
- مطابقة Call/Put مع اتجاه السهم.
- إضافة درجة المحفز الرسمي إلى العقد.
- أهداف ديناميكية مشتقة من حركة السهم وGreeks بدل نسب ثابتة عامة.

## حدود لا يمكن تجميلها

- Open Interest عادة لا يتحدث لحظيًا داخل الجلسة؛ Vol/OI إشارة لا إثبات.
- السعر القريب من ask لا يساوي Sweep مؤكدًا.
- Greeks المحسوبة محليًا تقريب Black–Scholes عند غياب Greeks من المزود.
- أخبار وموافقات FDA قد تحتاج مطابقة اسم الشركة بالرمز؛ لذلك يعرض النظام درجة الثقة والمصدر.
- أسعار المتابعة ليست إثبات تنفيذ ولا تثبت ترتيب الهدف والوقف بين فحصين.
- لا يوجد ضمان ربح، ولا إرسال خارجي، ولا تنفيذ آلي للأوامر.
